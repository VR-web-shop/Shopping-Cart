# Shopping Cart
Shopping Cart Microservice

## Install
1. Install Dependencies
```
$ npm install
```

2. Setup Environment Variables
```
$ cp .env.example .env
```

3. Seed Database
```
$ npm run seed
```

## Usage
```
npm start
```

## API Documentation
1. Start Server
2. Visit [localhost:3000/api/v1/documentation](http://localhost:3000/api/v1/documentation) for Swagger UI
